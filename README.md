## Innomeet Sample UI with React

## Few Steps

Open the terminal and type

```
// make directory
mkdir innomeet-sample-app
cd innomeet-sample-app

// clone
git clone https://github.com/mayuraitavadekar/innomeet-sample-app.git
npm install

// run 
sudo npm start

// open in the browser 
http://localhost:3000/

```
